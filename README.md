# Circle OX

เว็บเกม OX แบบวงกลมซ้อนขนาดสำหรับเล่นบนมือถือ และอัปโหลดขึ้น GitHub Pages ได้ทันที

## กติกา

- กระดาน 3x3
- ผู้เล่น 2 สี: แดง และน้ำเงิน
- แต่ละสีมีวงกลม 9 ตัว: ใหญ่ 3 กลาง 3 เล็ก 3
- วางทับได้เฉพาะตัวอีกฝ่ายที่เล็กกว่าเท่านั้น ห้ามทับตัวเอง
- วงใหญ่ทับวงกลางหรือวงเล็กของอีกฝ่ายได้ และเมื่อวงใหญ่อยู่บนสุดจะไม่มีอะไรทับต่อได้
- วงกลางทับวงเล็กของอีกฝ่ายได้ แต่ทับวงใหญ่ไม่ได้
- วงเล็กวางได้เฉพาะช่องว่าง
- ถ้าฝั่งที่ถึงตาไม่มีช่องให้วางได้ จะข้ามให้อีกฝั่งเล่น
- ถ้าทั้งสองฝั่งวางไม่ได้แล้ว เกมจะเสมอและเริ่มใหม่อัตโนมัติ
- นับผู้ชนะจากวงที่อยู่ชั้นบนสุด ใครเรียงครบ 3 ช่องก่อนชนะ

## วิธีเปิดเล่น

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที

## เล่นออนไลน์กับเพื่อนแบบฟรี

เกมนี้รองรับ Firebase Realtime Database ซึ่งใช้กับ GitHub Pages ได้

### ตั้งค่า Firebase

1. เข้าเว็บ Firebase Console แล้วสร้างโปรเจกต์ใหม่
2. สร้าง Web app ในโปรเจกต์นั้น
3. คัดลอก Firebase config
4. เปิดไฟล์ `firebase-config.js`
5. แทนค่า `null` ด้วย config ที่คัดลอกมา
6. เปิด Realtime Database แล้วเลือกโหมดทดสอบหรือเขียน rules ให้เว็บอ่าน/เขียนได้

Rules แบบง่ายสำหรับทดสอบ:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

Rules นี้เหมาะสำหรับทดลองเล่นเท่านั้น เพราะทุกคนที่มีลิงก์ฐานข้อมูลจะอ่าน/เขียนได้

หลังตั้งค่าแล้ว ในหน้าเกมจะใช้ปุ่ม:

- `เล่นปกติ` สำหรับเล่นบนเครื่องเดียวแบบทั่วไป
- `ต่อหน้า` สำหรับผลัดกันเล่นกับคนที่อยู่ด้วยกัน
- `สร้างห้อง` สำหรับเล่นออนไลน์ โดยผู้สร้างห้องเป็นผู้เล่นสีแดง
- `คัดลอกลิงก์` เพื่อส่งให้เพื่อน
- เพื่อนเปิดลิงก์แล้วจะเป็นสีน้ำเงิน
- หรือใส่รหัสห้องแล้วกด `เข้าห้อง`
- หน้าเกมจะแสดงว่าเพื่อนเข้าห้องแล้วหรือยัง
- เจ้าของห้องเลือกได้ว่าแดงหรือน้ำเงินเริ่มก่อน ก่อนมีการวางหมากตัวแรก

## Deploy บน GitHub Pages

1. สร้าง GitHub repository ใหม่
2. อัปโหลดไฟล์ `index.html`, `styles.css`, `script.js`, `firebase-config.js`, และ `README.md`
3. ไปที่ `Settings` > `Pages`
4. เลือก `Deploy from a branch`
5. เลือก branch `main` และ folder `/root`
6. กด `Save`

จากนั้นรอประมาณ 1-5 นาที GitHub จะให้ลิงก์เว็บสำหรับเปิดเล่นบนมือถือ
