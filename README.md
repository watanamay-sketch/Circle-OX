# Circle OX

เว็บเกม OX แบบวงกลมซ้อนขนาดสำหรับเล่นบนมือถือ และอัปโหลดขึ้น GitHub Pages ได้ทันที

## กติกา

- กระดาน 3x3
- ผู้เล่น 2 สี: แดง และน้ำเงิน
- แต่ละสีมีวงกลม 9 ตัว: ใหญ่ 3 กลาง 3 เล็ก 3
- วงใหญ่ทับวงกลางหรือวงเล็กได้ และเมื่อวงใหญ่อยู่บนสุดจะไม่มีอะไรทับต่อได้
- วงกลางทับวงเล็กได้ แต่ทับวงใหญ่ไม่ได้
- วงเล็กวางได้เฉพาะช่องว่าง
- นับผู้ชนะจากวงที่อยู่ชั้นบนสุด ใครเรียงครบ 3 ช่องก่อนชนะ

## วิธีเปิดเล่น

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที

## Deploy บน GitHub Pages

1. สร้าง GitHub repository ใหม่
2. อัปโหลดไฟล์ `index.html`, `styles.css`, `script.js`, และ `README.md`
3. ไปที่ `Settings` > `Pages`
4. เลือก `Deploy from a branch`
5. เลือก branch `main` และ folder `/root`
6. กด `Save`

จากนั้นรอประมาณ 1-5 นาที GitHub จะให้ลิงก์เว็บสำหรับเปิดเล่นบนมือถือ
